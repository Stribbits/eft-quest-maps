# EFT Quest Maps

Interactive Escape from Tarkov quest maps — all maps in one page with a switcher,
per-trader colour coding, show/hide + highlight toggles, zoom/pan, keys and
map-wide task lists.

Fully fact-checked: Lighthouse, Streets of Tarkov, Customs.
In progress: Woods, Shoreline, Interchange, Reserve, Ground Zero.

Maps: community SVGs (tarkov.dev / the-hideout). Quest data: EFT Wiki, TarkovHead,
GameMaps and community guides. Single self-contained `index.html` — no build step.
